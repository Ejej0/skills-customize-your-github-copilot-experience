
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman-style word game in Python that uses strings, loops, conditionals, and user input to create an interactive guessing experience.

## 📝 Tasks

### 🛠️ Build the Core Game Loop

#### Description
Use the starter code to create the main loop for the game. The player should be able to guess letters one at a time and see their progress as the hidden word is revealed.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses from the player
- Show the current progress in an underscore format, such as `_ _ _ _`
- Keep track of correct and incorrect guesses

### 🛠️ Add Win and Loss Conditions

#### Description
Finish the game by handling success, failure, and end-of-game messages so the player always knows when the game is over.

#### Requirements
Completed program should:

- End when the word is guessed or the player runs out of attempts
- Display the number of guesses remaining
- Show a win message when the player guesses the word correctly
- Show a lose message when the player uses all available attempts
